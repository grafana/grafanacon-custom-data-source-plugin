Road Weather Station API Server

This server mimics the data and api from Open Data Seattle's Road Weather Station API. This page contains documenation on how to access this mocked server as well as other helpful resources from the workshop.

On This Page:

  • Available Endpoints
  • Authentication
  • Workshop Commits

Available Endpoints:

All data:

http://localhost:8080/resource/egc4-d24i.json?$$app_token=good

Filter by station name:

http://localhost:8080/resource/egc4-d24i.json?$$app_token=good&stationname=JoseRizalBridgeNorth

Docker internal access:

http://host.docker.internal:8080/resource/egc4-d24i.json?$$app_token=good

Authentication:

All requests require the app token parameter: $$app_token=good

You can test failing authentication with: $$app_token=bad
